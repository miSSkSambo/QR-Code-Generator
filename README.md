A simple and beginner-friendly **Python application** that generates QR codes from text, URLs, phone numbers, email addresses, or any other input and saves them as PNG images.

---

## 📸 Preview

```text
========================================
        QR Code Generator
========================================

Enter text or URL:
https://github.com

Enter filename (without .png):
github

✅ QR Code saved as github.png
```

---

# ✨ Features

- ✅ Generate QR codes from text
- 🌐 Generate QR codes from URLs
- 📱 Generate QR codes for phone numbers
- 📧 Generate QR codes for email addresses
- 💾 Save QR codes as PNG images
- 📝 Custom filename support
- ⚡ Fast and lightweight
- 🐍 Beginner-friendly Python project

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/QR-Code-Generator.git
```

## Navigate into the Project

```bash
cd QR-Code-Generator
```

## Create a Virtual Environment (Optional)

```bash
python -m venv .venv
```

## Activate the Virtual Environment

### Windows

```bash
.\.venv\Scripts\activate
```

### macOS/Linux

```bash
source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Application

```bash
python main.py
```
